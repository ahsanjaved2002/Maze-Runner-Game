🧩 Maze Runner Game (C++)

A console-based Maze Runner game developed in C++. The maze is loaded dynamically from a text file, and the player must navigate through it while managing limited lives and moves to reach the destination.

🎮 Game Features

📂 Maze loaded from .txt file

🗺️ Dynamic 2D array using pointers (char**)

🎯 Three difficulty modes:

Easy (3 lives)

Medium (2 lives)

Hard (1 life)

👾 Maze Symbols:

* → Free path

| → Blocked path

+ → Monster

0 → Player position

❤️ Limited lives system

🔢 Limited moves system

🏆 Score calculation

📊 High score tracking (highrecord.txt)

🔁 Restart option after losing

🛠️ Concepts Used

File Handling (ifstream, ofstream)

Dynamic Memory Allocation

2D Dynamic Arrays

Pointers & References

Functions

Loops and Conditional Statements

Basic Game Logic Implementation
